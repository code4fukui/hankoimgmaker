# Hanko Image Maker


日本の判子（個人印）の画像を作成するための、シンプルなブラウザベースのツールです。テキストを入力し、角度を調整して、自分だけの判子を保存しましょう。

## デモ

**ライブデモ:** [https://code4fukui.github.io/hankoimgmaker/](https://code4fukui.github.io/hankoimgmaker/)

## 特徴

- **即時生成:** ブラウザ上で直接判子画像を生成します。インストールは不要です。
- **縦書き:** 1〜3文字を伝統的な縦書きスタイルで自動的に配置します。
- **回転の調整:** スライダーを使用して判子を傾けることで、より本物の手押しのような見た目にすることができます。

## 使い方

1.  **テキストの入力:** 入力フィールドに1〜3文字を入力します。画像は自動的に更新されます。
2.  **角度の調整:** 画像の下にあるスライダーを使用して、判子を回転させます。
3.  **画像の保存:** キャンバス上の生成された判子を右クリックし、「名前を付けて画像を保存...」を選択してダウンロードします。
