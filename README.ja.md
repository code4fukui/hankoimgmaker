# Hanko Image Maker

Hanko Image Maker は、ブラウザでオンラインで判子の画像を作成することができるウェブアプリケーションです。

## デモ
[Hanko Image Maker デモページ](https://code4fukui.github.io/hankoimgmaker/)

## 機能
- 1文字から3文字までの判子を作成できます
- 判子のサイズと角度を調整できます

## 使い方
ブラウザでデモページを開き、お好みの判子の文字を入力して、スライダーを動かして角度を調整すると、その場で判子の画像が生成されます。
