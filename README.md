# Hanko Image Maker

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A tool to create images of Japanese hanko (personal seals).

## Demo
https://code4fukui.github.io/hankoimgmaker/

## Features
- Create images of Japanese hanko (personal seals)
- Adjust the rotation angle of the hanko
- Support for 1, 2, and 3 character hanko

## Usage
Enter the text for the hanko in the input field and the image will be generated in the canvas below. Use the slider to adjust the rotation angle of the hanko.
