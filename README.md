# Hanko Image Maker

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple, browser-based tool for creating images of Japanese hanko (personal seals). Enter your text, adjust the angle, and save your custom seal.

## Demo

**Try it live:** [https://code4fukui.github.io/hankoimgmaker/](https://code4fukui.github.io/hankoimgmaker/)

## Features

- **Instant Generation:** Creates hanko images directly in your browser—no installation needed.
- **Vertical Text:** Automatically arranges 1, 2, or 3 characters in the traditional vertical style.
- **Adjustable Rotation:** Use the slider to tilt the seal for a more authentic, hand-stamped look.

## How to Use

1.  **Enter Text:** Type 1 to 3 characters into the input field. The image updates automatically.
2.  **Adjust Angle:** Use the slider below the image to rotate the hanko.
3.  **Save Image:** Right-click the generated seal on the canvas and select "Save image as..." to download it.
